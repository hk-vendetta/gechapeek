# Oathsworn: Compact - Gacha Guide (Player v2)

Last updated: 2026-04-08

## What This Covers
This guide explains banners, pull rates, pity, spark, duplicates, and the new attunement reveal flow.

## Summon Flow (What You See)
1. Attunement screen shows active banner details.
2. You summon.
3. Featured Prism reveal plays first (tap to unseal, crack, reveal, tap to continue).
4. Remaining pulls appear in the prism grid and auto-crack.
5. Close returns to attunement screen.

## Banner Info on Attunement Screen
The attunement screen can show:
- Banner Name
- Banner Description
- Featured Unit Summary
- Rate Summary

Example for featured_lyria_dungeon_path:
- Name: Path of the Vanguard - Lyria
- Featured Unit: Lyria
- Rate Summary: Tier 5 total rate, featured share, spark target

## Pull Costs
- Standard banner: 100 Compact Seals per pull
- Featured banner: 200 Compact Seals per pull

## Core Rates
Default tier distribution:
- Tier 1: 15%
- Tier 2: 30%
- Tier 3: 37%
- Tier 4: 15%
- Tier 5: 3%

Featured banners typically split Tier 5 into:
- 1.5% featured
- 1.5% non-featured Tier 5

## Pity and Spark
- Soft pity starts at pull 61 on featured flow.
- Hard pity threshold is banner-driven (commonly 80).
- Spark threshold is banner-driven (commonly 160).
- Spark can be claimed for featured units when threshold is met.

## Duplicate Value
Duplicates grant Prism Shards based on rarity and featured bonus rules. Shards are used for unit progression and are never wasted pulls.

## Practical Pull Advice
1. Pull featured banners if you are targeting one unit.
2. Track spark progress before swapping banners.
3. Use attunement banner summary text to confirm the active banner before spending.
4. If you only need roster depth, standard/all-units banners are more pull-efficient.

## FAQ
Q: Why does the reveal say Oath-Prism or Covenant Prism?
A: Both are presentation terms for the same summon infrastructure at different ritual tiers.

Q: Do I need to assign every banner text label in UI?
A: No. Any unassigned label is optional and safely ignored.
